# Aperture — Marketing UI Kit

Full-bleed, alternating-tile marketing site, cloned directly from `DESIGN-apple.md`'s section-rhythm system (light → dark → parchment → dark → footer). Copy is written for Aperture's actual product per the readme's Content Fundamentals section — no filler.

- `Nav.jsx` — 44px black global nav
- `ProductTile.jsx` — the one reusable section primitive (light/parchment/dark tone prop)
- `Footer.jsx` — dense Parchment footer
- `index.html` — assembled page using the design system's `Button`, `Tag`, `Card` components from `_ds_bundle.js`
